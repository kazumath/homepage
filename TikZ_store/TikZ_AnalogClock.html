TikZでアナログ時計

TikZでアナログ時計を書きました. コンパイルごとに現在時刻を取得してアナログ時計が描画されます(コンパイルはplatexで行ってください). アラビア数字とローマ数字の2種類を作りました.

コード

以下がコードです. ご自由にお使いください. 

\tikzset{  analog clock/.pic={      % パラメータ      \def\hr{\the\hour}      \def\mn{\the\minute}      \pgfmathsetmacro{\minuteAngle}{90 - \mn * 6}      \pgfmathsetmacro{\hourAngle}{90 - ( (mod(\hr, 12)) + \mn / 60 ) * 30}      \def\clockRadius{3} % 時計の半径      %針      \begin{scope}[line cap=round]        \draw[line width=5pt] (0,0)--(\hourAngle:{\clockRadius*0.5});         \draw[line width=3pt] (0,0)--(\minuteAngle:{\clockRadius*0.7});      \end{scope}      %外枠      \begin{scope}        \draw[double] (0,0) circle (\clockRadius);        \filldraw[gray] (0,0) circle (0.02*\clockRadius);        \node at(0,-\clockRadius*0.5) {\today};        \foreach \i in {1,2,...,12}{        \node at ({90-\i*30}:{\clockRadius*0.9}) {\Large\bfseries\i};        }      \end{scope}  },  analog clock roman/.pic={      % パラメータ      \def\hr{\the\hour}      \def\mn{\the\minute}      \pgfmathsetmacro{\minuteAngle}{90 - \mn * 6}      \pgfmathsetmacro{\hourAngle}{90 - ( (mod(\hr, 12)) + \mn / 60 ) * 30}      \def\clockRadius{3} % 時計の半径      %針      \begin{scope}[line cap=round]        \draw[line width=5pt] (0,0)--(\hourAngle:{\clockRadius*0.5});         \draw[line width=3pt] (0,0)--(\minuteAngle:{\clockRadius*0.7});      \end{scope}      %外枠      \begin{scope}        \draw[double] (0,0) circle (\clockRadius);        \filldraw[gray] (0,0) circle (0.02*\clockRadius);        \node at(0,-\clockRadius*0.5) {\today};        \foreach \i in {1,2,...,12}{        \pgfmathsetmacro{\ang}{\i*30}        \node at (90-\ang:{\clockRadius*0.9}) {\rotatebox{-\ang}{\large\bfseries\uppercase\expandafter{\romannumeral \i \relax}}};        }      \end{scope}  }}\centering\begin{tikzpicture}  \pic at (0,0) {analog clock};  \pic at (7,0) {analog clock roman};\end{tikzpicture}
