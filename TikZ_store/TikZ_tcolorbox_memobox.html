リングメモ風のtcolorboxによる枠

Xの投稿を見てつくったtcolorboxの枠です.

コード

以下がコードです. ご自由にお使いください. 使うときはプリアンブルに\usetikzlibrary{decorations.markings}を入れておいてください. 

\tikzset{memoframe/.style={gray,line width=1.5pt}}\newtcolorbox{memobox}[1]{    enhanced,breakable,    attach boxed title to top left={xshift=5mm,yshift=-\tcboxedtitleheight/2},    boxed title style={        enhanced,        empty,        sharp corners,    },    toprule=\tcboxedtitleheight/2,    empty,title=#1,    sharp corners,    drop shadow=teal!50!gray,    fonttitle=\bfseries,coltitle=teal!70!gray,    underlay boxed title={    \begin{scope}\clip ([xshift=-2pt]frame.north west) rectangle (frame.south east);    %%背景の方眼        \fill[white](frame.south west)rectangle(frame.north east);        \foreach \x in {0.5,1,...,30} {          \draw[cyan!20, opacity=0.6, line width=3pt] (\x,0) -- (\x,30);        }        \foreach \y in {0.5,1,...,30} {          \draw[cyan!20, opacity=0.6, line width=3pt] (0,\y) -- (30,\y);        }    %%穴をあける        \draw[memoframe,        postaction={            decorate,            decoration={            markings,            mark=between positions 0.03 and 0.97 step 15pt with {                \draw[white, line width=3pt] (0,0)--(4pt*1.41,0);                \filldraw[fill=white](0,0) arc[start angle=180+45,end angle=-45,radius=4pt];            }            }        }        ] (frame.north west) -- (frame.south west);    \end{scope}    \draw[memoframe](frame.south west)--(frame.south east)--(frame.north east)--(frame.north west);    \filldraw[thick,gray!60!white,fill=white](title.south west)--([xshift=-10pt]title.south east)    to[out=0,in=-160]++(10pt,2pt)--++(0,\tcboxedtitleheight)    to[out=-160,in=0]([xshift=-10pt]title.north east)--(title.north west)--cycle;    \filldraw[teal!70!gray]([xshift=-2pt]title.south west)rectangle++(5pt,\tcboxedtitleheight);    },}
